# ttpExtractor

## Install

```
$ pip install -r requirements.txt
```

## Commandline 

```
$ cp ttp-test.pdf input/
$ python ./ttpextractor.py ttp-test.pdf
```

Result is in: 
* output/ttp-test.pdf/

## Start Web UI

```
$ export OPENAI_API_KEY="..."
$ export UPLOAD_PW="..."
$ python ./web.py
```
